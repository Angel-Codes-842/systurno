# ⚡ Inicio Rápido

## Requisitos
- Python 3.10+
- Node.js 18+

## Instalación
```bash
./setup.sh
./start.sh
```

## URLs
- **Kiosko**: http://localhost:3000/kiosko
- **Turnos**: http://localhost:3000/turnos
- **Sala Espera**: http://localhost:3000/sala-espera

## Probar
1. Abrir Kiosko → Sacar turno
2. Abrir Turnos → Llamar turno
3. Ver en Sala Espera → Aparece el número

## Detener
```bash
Ctrl+C
```
