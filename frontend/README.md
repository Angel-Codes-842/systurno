# Frontend - Sistema de Turnos

React + TypeScript + Tailwind CSS

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
```

## Páginas

- `/kiosko` - Pantalla táctil para sacar turno
- `/turnos` - Panel para llamar turnos
- `/sala-espera` - Pantalla TV con turnos llamados
- `/panel` - Panel de especialistas (requiere login)

## Variables de Entorno

```env
VITE_API_URL=http://localhost:8000/api
VITE_WS_URL=ws://localhost:8000/ws
```
